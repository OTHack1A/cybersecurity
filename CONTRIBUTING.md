# Contributing to the Project

Thank you for your interest in improving this educational repository!

---

## How to Contribute

### 1. Fork & Clone

```bash
git clone https://github.com/<your-username>/cybersecurity.git
cd cybersecurity
```

### 2. Create a Branch

Use a descriptive name:

```bash
git checkout -b feature/add-wireshark-notes
```

### 3. Add Material

- Add PDFs, notes or resources in the appropriate folder
- Update `README.md` with a description of the new material
- Make sure the content is **for educational purposes only**

### 4. Commit & Push

```bash
git add .
git commit -m "docs: add notes on <topic>"
git push origin feature/add-wireshark-notes
```

### 5. Open a Pull Request

Open a PR describing:
- What you added or changed
- Why it could be useful to other students

---

## Guidelines

- Material must be **for educational purposes only** — no copyright violations
- All techniques must only be applied in **authorized environments**
- PDFs must be readable and of good quality
- Grammar corrections and improvements are always welcome
- Maintain a collaborative and inclusive tone

---

## Code of Conduct

Please maintain respectful and collaborative behavior.

---

*Every contribution, big or small, is valuable to the student community.*
